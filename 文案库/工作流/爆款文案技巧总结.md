## 一、文案结构模板

### 模板1：痛点+解决方案型
```
【开场】抛出痛点/疑问（前3秒抓住注意力）
【正文】分点讲解解决方案
【案例】具体产品/品牌型号
【对比】优缺点/价格对比
【结尾】总结推荐+CTA
```

### 模板2：榜单排名型
```
【开场】悬念开场（如：第X名居然是...）
【正文】从低到高排名讲解
【结尾】TOP1重点推荐+购买引导
```

### 模板3：教程攻略型
```
【开场】问题导入（你是不是也...）
【正文】步骤式讲解（1、2、3...）
【结尾】延伸阅读+关注引导
```

## 三、写作技巧库

### 1. 数字对比法
- 用具体数字制造冲击感
- 示例："活动期间仅需3000出头" vs "官网平时售价5000多"
- 示例："两千块价位居然做到了酒店同款"

### 2. 具体型号背书
- 品牌+具体型号增加可信度
- 示例："金可儿1898典藏版"、"栖作·序曲"
- 示例："威斯汀酒店'天梦之床'升级版"

### 3. 个人体验代入
- 用"我试过"、"我用过的"、"住过酒店"增加真实感
- 示例："我手里这款..."、"当初吸引我的就是..."

### 4. 场景化描述
- 让用户能想象到使用场景
- 示例："翻身起身不易惊动旁人，浅眠者也能睡得踏实"
- 示例："第二天醒来不会脖子酸"

### 5. 优缺点对比
- 诚实列出缺点反而增加信任
- 示例："但如果你腰椎不太好、需要更强支撑，坚果就更合适"
- 示例："唯一需要注意的是睡感偏软"

### 6. 限时/稀缺感
- 示例："活动价通常在4000多"
- 示例："活动期间仅需3000出头"

## 四、话术模式

### 引导关注
```
"关注我，稳赚不亏"
"需要的我们评论区见"
"不想花冤枉钱的朋友，直接抄作业就行"
```

### 促单话术
```
"这价格这睡感，还要啥自行车？"
"性价比相当突出"
"综合表现非常能打"
```

### 信任建立
```
"完全是我自己用过的分享"
"这期视频完全是我自己用过的枕头分享"
"我是护肤博主，不会有XX品牌找我打广告"
```
