# UI Designer Subagent - 使用示例

## 快速开始

当你需要设计一个界面时，可以这样与UI Designer交互：

```
请帮我设计一个[页面名称]，包含[功能描述]
```

例如：
```
请帮我设计一个登录页面，包含手机号+验证码登录
```

## 输出内容

UI Designer会输出三部分内容：

### 1. 设计规范文档
包含：
- 页面概述（目的、用户、场景）
- 视觉设计（色彩、字体、间距、圆角、阴影）
- 组件清单（所有状态）
- 交互规格
- 异常处理

### 2. 可运行React代码
使用：
- React + TypeScript
- TailwindCSS（Apple风格）
- Framer Motion（动效）
- CLSX（类名合并）

### 3. 交互规格说明
包含：
- 状态定义表格
- 动画参数
- 手势反馈
- 页面过渡

## 设计原则

遵循Apple HIG三大原则：
- **清晰**：内容为核心，视觉层次分明
- **服从**：交互服从于内容
- **深度**：层次感和动效增强理解

## 设计系统

### 色彩系统
- Primary: #007AFF
- Success: #34C759
- Warning: #FF9500
- Error: #FF3B30

### 间距系统（8px网格）
- xs: 4px, sm: 8px, md: 16px, lg: 24px, xl: 32px

### 圆角系统
- 按钮: 12px
- 卡片: 16px
- 模态框: 20px

### 阴影系统
- sm: `0 1px 2px rgba(0,0,0,0.05)`
- md: `0 4px 12px rgba(0,0,0,0.1)`
- lg: `0 12px 40px rgba(0,0,0,0.15)`

## 验证清单

输出后请检查：
- [ ] 颜色使用Apple色板
- [ ] 圆角在8px-20px范围
- [ ] 间距遵循8px网格
- [ ] 动效使用spring/ease-out
- [ ] 所有状态已覆盖
- [ ] 代码可运行无语法错误
