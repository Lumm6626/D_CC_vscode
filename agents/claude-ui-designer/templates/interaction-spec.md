# 交互规格说明

## 概述

本文档描述界面中所有交互元素的详细规格，包括状态定义、动画参数、手势反馈等。

## 状态定义

### 按钮状态

| 状态 | 触发条件 | 视觉表现 | 交互反馈 |
|------|----------|----------|----------|
| Default | 初始状态 | 正常显示，背景色 #007AFF，文字白色 | - |
| Hover | 鼠标悬停 | 背景亮度+5%，阴影加深，scale(1.02) | 鼠标变为pointer |
| Active | 鼠标按下 | 背景亮度-10%，scale(0.98) | 轻微触觉反馈 |
| Disabled | disabled=true | 50%透明度，cursor: not-allowed | 无交互 |
| Loading | loading=true | 显示spinner，禁止重复点击 | 无交互 |

### 输入框状态

| 状态 | 触发条件 | 视觉表现 | 交互反馈 |
|------|----------|----------|----------|
| Default | 初始状态 | 边框 #D2D2D7，内阴影 | - |
| Hover | 鼠标悬停 | 边框颜色变深 #86868B | 鼠标变为text |
| Focus | 输入框聚焦 | 边框 #007AFF，3px蓝色光圈 | 打开键盘 |
| Filled | 有输入内容 | 正常显示 | - |
| Error | 验证失败 | 边框 #FF3B30，3px红色光圈 | 显示错误提示 |
| Disabled | disabled=true | 灰色背景，cursor: not-allowed | 无交互 |

### 卡片状态

| 状态 | 触发条件 | 视觉表现 | 交互反馈 |
|------|----------|----------|----------|
| Default | 初始状态 | 轻微阴影，白色背景 | - |
| Hover | 鼠标悬停 | 上浮2px，阴影加深 | 鼠标变为pointer |
| Active | 鼠标按下 | 阴影恢复默认 | 轻微触觉反馈 |
| Pressed | 点击中 | scale(0.99) | 深度触觉反馈 |

## 动画规格

### 全局动画配置

```typescript
const animationConfig = {
  // 快速交互（微交互、状态切换）
  fast: {
    duration: 150,
    easing: 'cubic-bezier(0.4, 0, 0.2, 1)',
  },
  // 标准过渡
  normal: {
    duration: 250,
    easing: 'cubic-bezier(0.4, 0, 0.2, 1)',
  },
  // 慢速过渡（页面、大型元素）
  slow: {
    duration: 400,
    easing: 'cubic-bezier(0.4, 0, 0.2, 1)',
  },
  // 弹性动画
  spring: {
    type: 'spring',
    stiffness: 300,
    damping: 30,
  },
};
```

### 组件动画清单

| 组件 | 动画 | 时长 | 缓动 | 参数 |
|------|------|------|------|------|
| Button hover | scale | 150ms | ease-out | scale: 1 → 1.02 |
| Button active | scale | 100ms | ease-in | scale: 1.02 → 0.98 |
| Card hover | translateY + shadow | 200ms | ease-out | y: 0 → -2px |
| Input focus | ring | 200ms | ease-out | ring: 0 → 3px |
| Modal open | scale + opacity | 400ms | spring | scale: 0.95 → 1, opacity: 0 → 1 |
| Modal close | scale + opacity | 200ms | ease-in | scale: 1 → 0.95, opacity: 1 → 0 |
| Page enter | translateY + opacity | 400ms | ease-out | y: 20 → 0, opacity: 0 → 1 |
| Page exit | opacity | 200ms | ease-in | opacity: 1 → 0 |
| Toast enter | translateY + opacity | 300ms | spring | y: -20 → 0, opacity: 0 → 1 |
| Toast exit | opacity | 200ms | ease-in | opacity: 1 → 0 |

## 页面过渡

### 过渡类型

| 类型 | 进入 | 退出 | 适用场景 |
|------|------|------|----------|
| Fade | opacity 0→1 | opacity 1→0 | 模态框 |
| Slide | translateX/Y | translateX/Y | 页面切换 |
| Scale | scale 0.95→1 | scale 1→0.95 | 弹窗 |
| Combine | fade + slide | fade + slide | 页面切换 |

### 默认页面过渡

```
进入动画:
- translateY: 20px → 0px
- opacity: 0 → 1
- duration: 400ms
- easing: cubic-bezier(0, 0, 0.2, 1)

退出动画:
- opacity: 1 → 0
- duration: 200ms
- easing: cubic-bezier(0.4, 0, 1, 1)
```

## 异常交互

### 加载状态

| 场景 | 处理方式 | 反馈 |
|------|----------|------|
| 页面加载 | 显示骨架屏 | 保持布局稳定 |
| 数据提交 | 显示spinner + 禁用按钮 | 防止重复提交 |
| 图片加载 | 显示占位符 | 渐显加载 |
| 列表加载 | 显示骨架列表 | 位置预占 |

### 错误状态

| 场景 | 处理方式 | 反馈 |
|------|----------|------|
| 网络错误 | 显示错误提示 + 重试按钮 | 友好提示 |
| 表单验证 | 输入框下方显示错误信息 | 即时反馈 |
| 服务器错误 | 显示错误页面 + 联系支持 | 清晰引导 |

### 空状态

| 场景 | 处理方式 | 反馈 |
|------|----------|------|
| 无数据 | 显示空状态插图 + 说明文字 | 引导操作 |
| 无搜索结果 | 显示空状态 + 搜索建议 | 提供帮助 |
| 无网络 | 显示缓存数据 + 断网提示 | 说明情况 |
