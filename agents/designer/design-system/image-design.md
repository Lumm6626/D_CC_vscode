# Image Design System - 图片设计规范

> Apple 设计语言 × 家居路漫漫自媒体视觉体系
> 2026-06-02
> 数据来源：Apple Instagram @apple (37M粉, 1300+帖子) + Apple Newsroom + Apple 品牌标识指南 PDF + 各平台官方尺寸规范
> 交叉验证：`source-references.md`

## 核心来源

本图片设计规范的数据来源分为三类：

**① Apple 官方来源：**
- Apple Instagram @apple：实际分析了 Shot on iPhone 作品风格、Carousel 叙事模式、配色倾向、文字策略
- Apple Newsroom：页面布局、缩略图设计、文章排版
- Apple 品牌标识指南 PDF (logo_guidelines.pdf)：Logo 4种变体、最小尺寸、安全区规则

**② 平台官方规范：**
- 抖音/小红书/视频号/公众号各自的官方尺寸指南
- Instagram 官方尺寸（4:5 / 1:1 / 9:16）
- YouTube Creator Academy 缩略图3秒法则

**③ Apple 系统设计语言：**
- SF Pro + PingFang SC 字体配对（来自 Apple HIG Typography）
- 品牌色板（来自 Apple 品牌指南 + Mobbin 交叉验证）

> 本文件中的社交媒体平台数据来自各平台官方规范，并非从 Apple 社交媒体直接提取（Apple 社交媒体不使用自媒体风格的尺寸排版）。但 Apple Instagram 的视觉风格、配色、叙事方法是本文档的审美基准。

## 二、视频缩略图设计规范

### 核心原则 (Apple 极简风 + 家居路漫漫)

1. **大字+大图** - 文字不超过主视觉区域的 1/3
2. **3秒法则** - 用户看到缩略图的 3 秒内必须传达核心信息
3. **高对比度** - 文字与背景色差 ≥4.5:1（WCAG AA）
4. **焦点居中** - 主体/产品位于画面中心偏下 1/3 位置
5. **克制用色** - 单图不超过 3 种强调色

### 缩略图字体规范

- **字体**: 标题用 `PingFang SC Semibold`（中文）/ `SF Pro Display Bold`（英文）
- **字号**: 主标题 80-120px，副标题 40-60px（基于 1080px 基准）
- **行数**: 不超过 3 行，每行不超过 10 个字
- **对齐**: 左对齐（推荐）或居中对齐
- **字距**: 中文 0，英文 -0.03em 至 -0.05em

### 缩略图文案排版规则

```
┌──────────────────────────────┐
│                              │
│   ┌────────────────────┐     │
│   │  主标题（大号粗体）  │     │
│   │  副标题（小号中等）  │     │
│   └────────────────────┘     │
│                              │
│          [产品/人物主体]      │
│                              │
│       [标签/角标/元素]       │
└──────────────────────────────┘
```

- **文字安全区**: 距边缘至少 40px
- **标题位置**: 上方 1/3 区域内（不会被头像/互动按钮遮挡）
- **角标位置**: 右下角（不会被播放进度条遮挡）

### 缩略图状态

| 状态 | 设计要点 |
|------|----------|
| 默认 | 高亮主体 + 克制文字，不过度修饰 |
| 高点击 | 增加箭头/惊叹号/数字标号（"2选1"、"99%"）|
| 系列 | 统一模板底色，换图换数字 |
| 失败 | 主色为警示色，对比度拉满 |

## 三、信息图设计规范

### 信息图排版层级

```
层级 1: 大标题 (34-48pt, Semibold) → 读者第一眼看到
层级 2: 核心数据/数字 (48-80pt, Bold) → 视觉焦点
层级 3: 副标题/说明 (17-20pt, Medium) → 补充理解
层级 4: 脚注/来源 (12-13pt, Regular) → 辅助信息
层级 5: 品牌标识 (水印/Logo) → 不可见但必须存在
```

### 信息图尺寸

| 类型 | 推荐尺寸 | 说明 |
|------|---------|------|
| 竖版信息图 | `1080×1920` | 抖音/小红书全屏 |
| 方版信息图 | `1080×1080` | 小红书方版 |
| 长文章配图 | `1080×[比例]` | 公众号/知乎 |

### 数据可视化规则

- **图表用色**: 从 Apple 系统色板取色（`#007AFF` `#5856D6` `#34C759` `#FF9500`）
- **数据标注**: 关键数据用 `#FF3B30` 或 `#007AFF` 高亮
- **渐进色**: 从浅到深渐变，保持 Apple 色板色调
- **避免**: 3D 图表、过度渐变、霓虹色

## 四、品牌视觉素材

### Logo 变体

| 变体 | 格式 | 尺寸 | 适用场景 |
|------|------|------|---------|
| 主Logo（浅底） | PNG/SVG | 透明背景 | 浅色页面/视频内 |
| 主Logo（深底） | PNG/SVG | 白色反转版 | 深色页面/视频内 |
| 图标Logo（favicon） | PNG | 32×32, 180×180 | 网站/App |
| 水印Logo（半透明） | PNG | `600×200`（透明度 30%）| 视频角标/图片水印 |
| 头像版 | PNG | `400×400` | 各平台头像 |

### Logo 使用规则

- **最小尺寸**: 不大于画面宽度的 20%，不小于 60px
- **位置**: 右下角（推荐）、左上角
- **留白**: Logo 周围保持 ≥ 其高度 30% 的留白
- **不要**: 拉伸变形、加描边、加投影、加渐变

### 图标系统

使用 SF Symbols 风格（Apple 系统图标标准）：
- **线框图标**: 2px 线宽，圆角端点
- **实心图标**: 与线框版互补，用于选中态
- **尺寸规范**: 24×24, 32×32, 48×48, 64×64
- **用色**: 使用 Apple 色板中对应语义色

## 五、图片+文字排版指南

### 图文混排规则 (Apple 风格)

```
好 → ┌──────────────┐
     │  文 字 区     │  ← 标题/说明在上方
     │              │
     │   图 片       │
     └──────────────┘

更好→ ┌──────────────┐
      │   图 片       │
      │              │
      │  文 字 区     │  ← 文字在下方（沉浸感更强）
      └──────────────┘

最推荐→┌──────────────┐
       │  图 文 融 合   │  ← 文字直接叠加在图片上
       │  [主视觉]     │     Apple 官网风格
       │  文字叠在图上  │
       └──────────────┘
```

### 文字叠加图片规范

- **遮罩层**: 图片上叠加 `rgba(0,0,0,0.2~0.4)` 渐变，确保文字可读
- **浅色图片**: 深色文字 (`#1D1D1F`)
- **深色图片**: 白色文字 (`#FFFFFF`)
- **文字区背景**: 可选 `backdrop-filter: blur(10px)` 毛玻璃背景带
- **安全区**: 文字保持在画面 80% 安全区内

### 图文间距规则

| 关系 | 间距 | 说明 |
|------|------|------|
| 标题→图片 | 16-24px | 8px 网格 × 2-3 |
| 图片→正文 | 16-24px | |
| 图片→图片 | 8-16px | 图文间隔 |
| 段落→段落 | 24px | |
| 正文→CTA按钮 | 32-48px | 转化要素 |

## 六、素材格式与输出标准

### 图片格式选择

| 格式 | 适用场景 | 备注 |
|------|---------|------|
| **PNG** | Logo、图标、截图、需要透明背景 | 无损，文件稍大 |
| **JPG** | 产品图、实拍照片、封面 | 有损，文件小，适合上传 |
| **WebP** | 网页使用 | 更小文件，Chrome/Edge 支持 |
| **SVG** | Logo、图标、插图 | 矢量化，无限缩放 |
| **GIF** | 小动画、微动效 | 不推荐，推荐用视频代替 |

### 输出质量标准

- **分辨率**: 不低于 72 DPI（适用于屏幕展示）
- **色彩空间**: sRGB（不要用 Adobe RGB / ProPhoto RGB）
- **文件大小**: 单张 ≤ 5MB（各平台上传限制）
- **压缩**: 使用主流压缩工具，避免可见压缩伪影
- **安全边框**: 重要内容至少距边缘 40px

### 文件命名规范

```
[用途]_[内容描述]_[日期].扩展名

示例：
封面_床垫选购指南_260602.jpg
信息图_床垫硬度对比_260602.png
缩略图_鸿蒙隐藏功能_260602.jpg
logo_家居路漫漫_主.png
logo_家居路漫漫_水印.png
banner_千禧光_产品页_260602.jpg
```

## 七、设计检查清单

### 社交媒体配图
- [ ] 尺寸符合对应平台标准
- [ ] 3:4 比例优先，文字在安全区内
- [ ] 对比度满足 WCAG AA (4.5:1)
- [ ] 缩略图 3 秒内能看懂内容
- [ ] 不超过 3 种强调色

### 品牌素材
- [ ] Logo 有浅底/深底版本
- [ ] 水印透明度 ≤ 30%
- [ ] 图标风格统一（SF Symbols 风格）
- [ ] 品牌色使用正确

### 排版
- [ ] 图文层级清晰（大标题→数据→说明→脚注）
- [ ] 文字叠加图片时有遮罩
- [ ] 间距遵循 8px 网格
- [ ] 字体使用 `PingFang SC` / `SF Pro`

### 输出
- [ ] sRGB 色彩空间
- [ ] 单文件 ≤ 5MB
- [ ] 命名规范
