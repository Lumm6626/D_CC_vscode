# Allergy Clinics 日报 Web 交互页面 - 设计需求文档

## 1. 项目概述

**项目名称：** Allergy Clinics 行业日报 Web 平台
**项目类型：** 行业新闻聚合 + 过敏专科资讯 + 交互式浏览
**核心价值：** 为过敏专科医生/诊所提供每日行业资讯汇总，过滤高相关性内容

---

## 2. 用户角色

| 角色 | 需求 |
|------|------|
| 过敏专科医生 | 快速浏览过敏相关资讯，追踪行业动态 |
| 诊所运营者 | 了解行业政策、市场趋势、竞争对手动态 |
| 医药从业者 | 跟踪过敏药物、治疗方案最新进展 |

---

## 3. 功能需求

### 3.1 核心功能（必须）

#### 3.1.1 新闻浏览
- 展示当日汇总的新闻列表
- 支持按地区（美国/欧洲/中国/澳大利亚/其他亚太）筛选
- 支持按来源（EAACI/AAAAI/ACAAI/ASCIA/Medscape等）筛选
- 支持按内容类型（过敏相关/行业资讯）筛选
- 过敏相关内容**高亮标注**

#### 3.1.2 搜索功能
- 关键词全文搜索
- 支持中英文
- 搜索结果高亮关键词

#### 3.1.3 邮件订阅
- 用户设置订阅偏好（地区偏好、过敏关键词定制）
- 每日/每周订阅频率选择
- 订阅管理（开启/暂停/取消）

### 3.2 增强功能（建议）

#### 3.2.1 个性化推荐
- 根据浏览历史推荐相关资讯
- 收藏功能

#### 3.2.2 数据可视化
- 新闻来源分布饼图
- 地区分布地图
- 过敏关键词趋势图

## 6. 页面结构设计建议

### 6.1 整体布局
```
┌──────────────────────────────────────────────────────────┐
│  Logo    搜索框                    订阅设置 | 用户登录   │  ← 固定Header
├──────────────────────────────────────────────────────────┤
│  筛选栏：地区 | 来源 | 类型 | 日期                       │  ← 工具栏
├──────────────────────────────────────────────────────────┤
│                                                          │
│  ┌─────────────────┐  ┌─────────────────────────────┐   │
│  │ 统计概览        │  │ 过敏重点关注 (卡片网格)     │   │
│  │ • 总数/过敏数   │  │                             │   │
│  │ • 地区分布      │  │                             │   │
│  └─────────────────┘  └─────────────────────────────┘   │
│                                                          │
│  ┌─────────────────────────────────────────────────────┐│
│  │ 新闻列表 (瀑布流/时间线)                              ││
│  │ • 过敏相关 - 橙色高亮卡片                            ││
│  │ • 行业资讯 - 蓝色普通卡片                            ││
│  └─────────────────────────────────────────────────────┘│
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 6.2 响应式断点
- **桌面端:** > 1200px (三栏/两栏布局)
- **平板端:** 768px - 1200px (两栏布局)
- **移动端:** < 768px (单栏，筛选折叠)

## 7. 视觉风格建议

- 标题: 思源黑体 / Microsoft YaHei

### 7.3 卡片风格
- 圆角: 8-12px
- 阴影: 0 2px 8px rgba(0,0,0,0.08)
- 边框: 左侧3px彩色边条标识类型

## 8. 交互设计要点

### 8.1 筛选交互
- 筛选条件以**标签(pill)** 形式展示
- 选中状态明显（填充色 vs 描边色）
- 支持多选
- 筛选结果**实时更新**，无刷新

### 8.2 新闻卡片交互
- 悬停：轻微上浮 + 阴影加深
- 点击：展开详情/跳转原链接
- 过敏标记：左上角红色角标

### 8.3 订阅设置
- 弹窗/侧边抽屉形式
- 表单式配置
- 保存后显示成功提示
