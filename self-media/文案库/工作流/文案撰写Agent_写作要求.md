## 二、开场钩子（前3秒）

开场钩子是留住观众的关键。必须在前3秒内制造"不得不继续看下去"的张力。

### 六大钩子类型

#### 1. 痛点型
**公式**：戳中一个具体痛苦 + 承诺解决
**示例**：
- "不会选床垫的朋友，买回来发现腰疼的有没有？"
- "华为手机这个功能不打开，你以为在用五六千的手机，其实只发挥了3000块的性能。"

#### 2. 疑问型
**公式**：抛出具体疑问 + 悬念感
**示例**：
- "3000元和1万元的床垫，到底差在哪里？"
- "华为出厂默认关掉了一个功能，开了效率翻倍，是哪个？"

#### 3. 悬念型
**公式**：制造期待 + 给出理由
**示例**：
- "2026年最值得买的床垫排行榜来了！历时一年对比了几百款，结论和我想象的完全不一样。"
- "鸿蒙6.0悄悄上线了5个宝藏功能，知道的人都在偷偷用。"

#### 4. 对比型
**公式**：制造反差 + 激发好奇心
**示例**：
- "床垫界公认的大牌就那几个，但价格嘛……懂的都懂。今天不聊贵的，专门挖性价比战神。"
- "别人花8000买的床垫，我用十分之一的价格搞定了，体验反而更好。"

#### 5. 数字型
**公式**：具体数字 + 冲击力
**示例**：
- "历时一年，对比了20多个品牌数百款床垫，这3款闭眼入不亏。"
- "6分钟开发完成，184个测试全部通过，这效率绝了。"

#### 6. 身份型
**公式**：强调专业身份 + 降低戒心
**示例**：
- "我，一个每年花10万拆床垫的资深家具人，今天来交作业。"
- "作为一个用过30多款床垫的人，这几款我必须说实话。"

## 三、正文结构模板

### 模板1：痛点+解决方案型（最适合家居产品）

```
【开场】抛出痛点/疑问（前3秒）
【背景】说明为什么这个问题重要
【正文】分点讲解解决方案（每个痛点对应一个解答）
【案例】具体品牌/型号/价格支撑
【对比】优缺点诚实分析
【结论】总结推荐 + 适合人群
【CTA】引导互动/关注
```

**适用场景**：床垫选购、手机技巧、对比评测类内容

---

### 模板2：榜单排名型（最适合盘点类内容）

```
【开场】悬念开场（"第X名居然是..."）
【规则】简单说明评选标准
【正文】从低到高排名讲解，每款：
  - 核心卖点（一句话说清）
  - 具体配置/参数
  - 适合人群
  - 价格/性价比
【结尾】TOP1重点推荐 + 购买建议
【CTA】评论区互动
```

**适用场景**：排行榜、年终盘点、榜单分析

---

### 模板3：教程攻略型（最适合数码/AI技巧类）

```
【开场】问题导入（"你是不是也..."）
【目录】预告本期内容（"3个技巧，第2个最容易被忽略"）
【正文】步骤式讲解
  - 每个步骤：操作路径 + 效果说明 + 适用场景
【延伸】补充注意事项
【总结】收藏引导 + 互动问题
```

**适用场景**：鸿蒙技巧、App推荐、AI工具教程

---

### 模板4：个人体验型（最适合种草/测评）

```
【开场】建立身份（用过的、拆过的、对比过的）
【体验】描述真实使用感受（好在哪/哪里一般）
【对比】和同类产品相比的优势/不足
【推荐】适合谁买/什么时候买
【CTA】"需要的评论区见"
```

**适用场景**：床垫体验、数码产品开箱、家居好物分享

## 四、结尾CTA写法

### 引导关注
```
"关注我，稳赚不亏"
"需要的我们评论区见"
"不想花冤枉钱的朋友，直接抄作业就行"
"还有什么想让我测的，评论区安排"
```

### 促单话术
```
"这价格这睡感，还要啥自行车？"
"性价比相当突出"
"综合表现非常能打"
"闭眼入不亏"
```

### 信任建立
```
"完全是我自己用过的分享"
"这期视频/内容是我自己用过的真实反馈"
"我是家居博主，不会有XX品牌找我打广告"
"说句得罪品牌的话……"
```

### 互动引导（开放式问题）
```
"你用华为多久了？还发现过什么隐藏功能？"
"你们家用的什么床垫？多少钱买的？"
"评论区说说你们的选购经历，抄作业来了"
```

## 五、语言风格规范

### 语气
- **必须**：口语化、像朋友推荐
- **必须**：敢说真话，敢列缺点
- **禁止**：过度营销语气（"家人们谁懂啊"）
- **禁止**：假大空（"极致奢华至尊体验"）

### 句式
- 短句为主，每句不超过20字
- 适当使用感叹句增强感染力
- 多用"我试过"、"我发现"、"说实话"
- 数字要具体（"3厘米乳胶"而不是"优质乳胶"）

### 用词
- **推荐**：具体型号、品牌名、具体数字、专业术语（特拉雷乳胶、独立袋装弹簧、鸿蒙超级终端）
- **避免**：模糊词（"非常好用"、"特别棒"）
- **避免**：绝对化宣传（"全球最好的"、"第一名的"）

### 禁忌
- 不说"家人们"
- 不说"OMG"
- 不说"姐妹们"
- 不夸大效果（"用了这个手机速度提升100倍"）

## 六、平台差异（抖音/小红书/视频号）

### 抖音
| 要素 | 要求 |
|------|------|
| 节奏 | 快，冲突强，每15秒一个信息点 |
| 开场 | 必须在前3秒制造强钩子 |
| 互动 | 结尾抛开放式问题，引导评论 |
| 标签 | 5-10个精准标签 |
| 字幕 | 必须有，方便静音观看 |
| 时长 | 1-3分钟为主 |

### 小红书
| 要素 | 要求 |
|------|------|
| 封面 | 标题要长，信息量大，有获得感 |
| 结构 | 干货先行，收藏价值高 |
| 互动 | 收藏引导、评论区交流 |
| 标签 | 10个以上，覆盖热门搜索词 |
| 排版 | 分点清晰，步骤明确 |
| 风格 | 实用主义，诚实测评 |

### 视频号
| 要素 | 要求 |
|------|------|
| 风格 | 情感共鸣优先于信息密度 |
| 节奏 | 比抖音稍慢，给观众反应时间 |
| 互动 | 引导转发给家人朋友 |
| 话题 | 贴近生活场景（中老年熟悉的话题） |
| 时长 | 可以稍长（3-5分钟） |

## 七、质量检查清单

文案完成后，必须逐项检查以下10条：

- [ ] **1. 开头强钩子**：前3秒是否有吸引力？能不能让人停下来？
- [ ] **2. 平台适配**：抖音/小红书/视频号版本是否已区分？
- [ ] **3. 具体数字**：是否有具体型号、价格、参数支撑？（不说"优质"，说"3厘米乳胶"）
- [ ] **4. 真诚度**：是否坦承缺点？有没有过度吹捧？
- [ ] **5. 口语化**：是否像朋友说话？有没有书面语？
- [ ] **6. 互动引导**：结尾有没有抛问题引导评论？
- [ ] **7. CTA明确**：有没有引导关注/收藏/评论？
- [ ] **8. 标签质量**：是否覆盖了精准话题标签？
- [ ] **9. 无禁忌词**：有没有"家人们"/"OMG"/绝对化宣传？
- [ ] **10. 核心信息清晰**：看完能记住一件事吗？

## 八、爆款文案公式

```
爆款文案 = 痛点/悬念开场 + 具体数字 + 专业术语背书 + 个人体验 + 真诚推荐 + CTA引导
```

### 公式拆解
1. **开场（前3秒）**：选一种钩子类型，用最强的
2. **内容（中段）**：信息量要足，每点都有具体支撑
3. **个人体验**：增加真实感，"我试过"、"我拆过"
4. **真诚度**：适当提缺点，反而增加信任
5. **结尾**：总结 + 开放式互动问题
