# 自媒体多平台发布助手

写一篇文案，自动生成**抖音、小红书、B站、视频号、快手、YouTube**六个平台的标题、话题标签和封面图。

## 封面工作流程

封面生成分两步：

```
[生成主封面] ──→ [可编辑] ──→ [裁剪各平台尺寸]
  --master          PS/Canva      --adapt
```

你不需要手动裁剪，工具会自动从主封面裁出各平台需要的尺寸。

## 用法

### 一、全流程（标题+话题+封面）

```bash
python -m publish_helper "文案输出\定稿_家居_床垫多少钱买才不交智商税_260427.txt"
```

依次完成：生成标题/话题 → 生成竖屏主封面 → 裁剪各平台封面

### 二、分步流程（推荐）

```bash
# 第1步：先看看标题和话题
python -m publish_helper "文案.txt" --text-only

# 第2步：生成竖屏主封面（1080×1920）
python -m publish_helper "文案.txt" --master

# 第3步：用PS/Canva编辑主封面，然后裁剪各平台尺寸
python -m publish_helper "文案.txt" --adapt "封面图\主封面\主封面_xxx.jpg"

# 或者一次完成：生成主封面 + 裁剪
python -m publish_helper "文案.txt" --cover-only
```

### 三、其他命令

```bash
python -m publish_helper --list          # 查看各平台规则
python -m publish_helper "文案.txt" --info  # 只分析文案
```

## 输出文件

| 输出 | 位置 | 命名规则 |
|------|------|---------|
| 标题+话题方案 | `发布方案/发布方案_xxx.txt` | - |
| 竖屏主封面 | `封面图/主封面/主封面_xxx.jpg` | 9:16，供定稿编辑 |
| 各平台封面 | `封面图/<文案名>/` | `平台_比例_文案名.jpg` |

## 各平台封面尺寸

| 平台 | 裁剪基准 | 生成尺寸 |
|------|---------|---------|
| 抖音 | 竖屏主封面 | 1080×1920（9:16）+ 1080×1440（3:4 feed） |
| 小红书 | 竖屏主封面 | 1080×1440（3:4）+ 1080×1920（9:16）+ 1080×1080（1:1） |
| B站 | 竖屏主封面 | 1920×1080（16:9）+ 1440×1080（4:3） |
| 视频号 | 竖屏主封面 | 1080×1920（9:16）+ 1080×1260（6:7 feed） |
| 快手 | 竖屏主封面 | 1080×1920（9:16） |
| YouTube | 竖屏主封面 | 1280×720（16:9） |

## 裁剪策略

从竖屏主封面自动裁剪各比例：
- **9:16** → 同比例缩放
- **3:4** → 裁顶部（保留标题文字）
- **6:7** → 裁顶部
- **1:1** → 居中裁正方形
- **16:9** → 居中裁横屏区域

## 自定义

- **封面配色**: `images.py` 中的 `UNIFIED_STYLE`
- **标题模板**: `titles.py` 中的 `TITLE_TEMPLATES`
- **标签模板**: `titles.py` 中的 `TAG_TEMPLATES`
- **平台规则**: `platform.py` 中的 `PLATFORMS`
